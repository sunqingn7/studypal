'''
A study pal program for all
'''

# what is it

This program is a study pal program, which can be a standalone app runs in any major platform, Macos, android, inux and Windows. also it has a webpage frontend.

# features

It initially contains 3 types of views:
 - File view, this is the main view which can open the study material in any format: txt, pdf, epub, website and so on
 - Note view: this is the place user can input note
 - AI view: this is the place user can ask questions about the file in display and the note he takes, and get helps from selected AI. 
and it will suppot plugin system to add more type of views, for example, possible a mindset view to support mindm naps.

The GUI initially show those 3 views, one for each type, layout main view is the file view, and in right side panel, top is AI view and button is note view.

User can 
 - add more views 
 - layout those views freely.

# Usage

This is mainly for study or research. User can open any document in the file view, write down anything in their mind in note view and ask questions to AI about what displayed in file view and notes in note view.

AI is configurable to connect to any provider:
 - online providers like openai, claude, ...
 - local providers like ollama, llama.cpp, vllm, SGLang ans so on
